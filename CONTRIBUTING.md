# Contributing to Secuditor Lite

Thank you for your interest in contributing to **Secuditor Lite**, Contributions such as testing, ideas, and feedback are welcome.

---

## Code of Conduct & Ethics

By contributing, you agree to uphold the following principles:

* All contributions must support **legal, ethical, and authorized diagnostics** only
* Being respectful of differing opinions, viewpoints, experiences, and Feedback
* The maintainer reserves the right to reject contributions that conflict with these principles

---

## Ways to Contribute

You can help in the following ways:

### Testing

* Test on different Windows versions
* Report bugs, regressions, or edge cases
* Validate accuracy, stability, and performance

### Ideas & Feedback

* Feature requests (ethical only)
* UI improvements for performance and output
* Suggestions for better logging or reporting

---

## Reporting Bugs

When reporting a bug, please include:

* Windows OS version
* Expected vs actual behavior
* Error output or logs (if available)
